What is HTML?

  • HTML Stands for Hyper Text Markup Language .
  • HTML describes the structure of web pages .
  • HTML is the standard markup language for creating Web Pages .
  • HTML consists series of element

Example explained : -

  • Doctype html tells us we are working on html 5
  • html root element of html pages
  • meta charset = "utf -8 " is basically font style (this include many of font type in it.)
  • meta name = viewport aspect ratio

tags
